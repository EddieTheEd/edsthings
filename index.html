Ed's Things

Hi! This is a public collection of the coding projects I have embarked on. Some of them don't work well (voice assistant), some of them are riddled with bugs (Frengraph). But overall they are pretty cool! Enjoy!

Current project: Nothing!! Yay!!



Ed's Obsidian Notes

A personal notes website, running with Blaze.

Frengraph

A fun graphing project for friendship groups.

Voice Assistant

A silly voice assistant in Python powered (?) by OpenAI.

Blaze

A simple static site generator for Github Pages written in Rust (Ossac, Tag).

Pinyinification

A lightweight translation tool for Chinese to English, with some Japanese functionality. Uses Google Translate API.

Demeter

A study tool, using Google Docs viewer.

China

Convert a .txt file of Chinese vocabulary to a Listening, Reading and Writing anki. Uses Google Translate and Google TTS.

Dating

A school tool to convert a school date to a calendar date.

The Lit Anki

Convert literature terms into an anki. (anonymoof1528)
